# 3D FPS Night Street Scene

## Project Description

A 3D FPS night street scene.

The scene includes:
- a highway with residential buildings on both sides;
- road lamps with night lighting;
- a controllable FPS player;
- a civilian car;
- a night sky with the moon;
- distant mountains on the horizon;
- background music.

The project is used to practice scene composition, lighting, atmosphere creation, and audio integration in a 3D environment.

## Project Goals

- Practice building 3D scenes
- Work with night lighting and light sources
- Create an atmospheric environment
- Integrate audio into a game scene
- Practice FPS perspective setup

## Scene Format

- Scene type: 3D
- Camera: First Person (FPS)
